# PDF Print Preview Util

https://aarich.github.io/pdf-print-preview

A tiny app for editing PDFs by adding/removing blank pages and previewing double-sided printing. Useful for preparing documents to print as booklets or checking page layout.

## Run Locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start dev server:

   ```bash
   npm run dev
   ```

3. Open the URL shown in terminal (typically http://localhost:5173)

4. Drag a PDF onto the page to begin editing.

5. Use the buttons between thumbnails to add blank pages to adjust the layout for double-sided printing.
