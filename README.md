# PDF Editor

A web app for editing PDFs by adding/removing blank pages and previewing double-sided printing. Useful for preparing documents to print as booklets or checking page layout.

## Run Locally

1. Install dependencies:

    ```bash
    npm install
    ```

2. Start dev server:

    ```bash
    npm run dev
    ```

3. Open the URL shown in terminal (typically http://localhost:5173)

4. Drag a PDF onto the page to begin editing

## Deploy to GitHub Pages

1. Push to main branch
2. Go to Settings → Pages → Source: GitHub Actions
3. The workflow will auto-deploy on every push
